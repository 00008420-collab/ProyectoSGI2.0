# GAPC - Sistema de Gestión (Streamlit)

Este repo contiene la app Streamlit para gestionar Miembros, Grupos, Préstamos, Pagos, Multas, Reuniones y un módulo avanzado. 

## Pasos rápidos
1. Subir este repo a GitHub.
2. Crear MySQL en Clever Cloud e importar tus tablas (phpMyAdmin).
3. En Streamlit Cloud crear una App apuntando al repo (`app.py`).
4. Añadir Secrets en Streamlit Cloud (TOML):
   - DB_HOST, DB_PORT, DB_USER, DB_PASS, DB_NAME
   - AUTH_MODE, AUTH_TABLE, AUTH_USER_COL, AUTH_PASS_COL, PASSWORD_HASH
5. Deploy.
